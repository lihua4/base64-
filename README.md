# Base64 图形化加解密工具 🛡️
 一款 **23.05KB极致迷你** 的 Base64 加解密安卓工具，Python 开发，图形化界面操作，支持结果一键复制到剪贴板～ 要知道现在手机随手拍一张照片，哪怕是压缩后的JPEG都普遍有500KB - 2MB，就连普通网页用的低分辨率图片都要100KB左右，这工具比绝大多数图片都小得多，几乎不占用手机存储空间。
 ## 核心亮点 ✨
 - 🐍 开发语言：Python（源码公开可查）
 - 🖥️ 交互方式：图形化界面（GUI），操作简单
 - 🔒 核心功能：Base64 加密 + 解密
 - 📋 便捷操作：加解密结果一键复制到剪贴板
 - 🪶 **极致迷你**：仅23.05KB，远小于常规图片体积
 - 📱 适配信息：仅安卓版，最低支持安卓5.0，目标适配安卓10
 ## 重要说明 ⚠️
 本 APK 由 **Python 编译器 IDE** 生成，需先在安卓设备上安装该应用（小米应用商店、荣耀应用市场等主流平台均可直接搜索下载），才能正常运行。
 ## 快速使用 🚀
 1.  打开小米应用商店、荣耀应用市场等平台，搜索并安装「Python 编译器 IDE」；
 2.  下载仓库中的 `Base64-Crypto-Tool.apk` 文件；
 3.  安装工具（安卓设备需开启“允许安装未知来源应用”权限）；
 4.  打开工具输入待加解密文本，点击「加密」或「解密」；
 5.  结果生成后点击「复制」，即可拷贝到剪贴板。
 ## 源码说明 📄
 仓库已上传核心 Python 源码文件（如 `base64_gui.py`），可查看加解密逻辑与 GUI 实现，支持自由二次开发。
